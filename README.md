# queuing-systems

## Requirements

- Python 3.8.5+

## How to run

1. Clone this repo to the desired directory

1. To install required packages run:

    ```Bash
    pip3 install -r requirements.txt 
    ```

    or

    ```Bash
    make reqs
    ```

1. To run the code use:

    ```Bash
    python3 main.py
    ```

    or

    ```Bash
    make start
    ```
